### Install Tensorflow


```
$ mkdir -p $TMPDIR
```

```
$ export TMPDIR=$HOME/tmp
```

```
$ TMPDIR=tmp pip3 install <package>
```

```
$ pip3 install -b $HOME/build
```

```
$ pip3 install --no-cache-dir tensorflow
```

### Install packages


```bash
$ (env) pip3 install --no-cache-dir flask flask-sqlalchemy pillow tensorflow
```
